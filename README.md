# Alura Store Challege
#### Desafio de Alura Latam
## Descripción del challenge
El desafío se trata de ayudar al Sr. Juan a decidir qué tienda de su cadena Alura Store debe vender para iniciar un nuevo emprendimiento. Para ello, se anala datos de ventas, rendimiento y reseñas de las 4 tiendas de Alura Store. El objetivo es identificar la tienda menos eficiente y presentar una recomendación final basada en los datos.

## Documentos del repositorio
- `READMED.md`: Descripción del contenido del repsitorio.
- `base-de-datos-callenge1-latam`: Contenido de las bases de datos de las distintas tiendas.
- `Alura_Store_Challenge.ipynb`: Codigo fuente del análisis.

## Tareas del challenge (Fuente: Alura LATAM)
- Cargar y manipular datos CSV con la biblioteca Pandas.
- Crear visualizaciones de datos con la biblioteca Matplotlib.
- Analizar métricas como ingresos, reseñas y rendimiento de ventas.
- Elaborar una recomendación y conclusión final fundamentada por el análisis.
